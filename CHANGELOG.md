## [1.0.0] - 2024-12-07

### Added
- Initial release of VAE Image Generation project
- Core VAE model implementation with encoder-decoder architecture
- Training script with basic features
- Image generation script
- Latent space interpolation
- Jupyter notebook for interactive use
- Utility functions for visualization
- Configuration file
- README documentation
- Requirements file
- License file

### Features
- Variational Autoencoder architecture
- Probabilistic latent space
- Reparameterization trick
- KL divergence regularization
- Support for MNIST and CIFAR-10 datasets
- Image generation from latent space
- Latent space interpolation
- Basic training and inference scripts
